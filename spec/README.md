# /spec

The SPICED protocol. This directory is the part WbD owns in the open: the field contract every skill and agent reasons against.

- `spiced.schema.json` the machine-readable SPICED record (the five namespaces).
- `namespaces.md` the definition of S, P, I, CE, D and the cross-cutting group.
- `SKILL.spec.md` the SKILL.md format contract.
