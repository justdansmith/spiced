# Project: Example Acme (illustrative)

> Reference only. This public repo never contains real customer data, names, or pulled records.

A project is a customer-shaped bundle: it selects which skills and agents deploy, and declares where compute and data live. See `project.yaml`.

The pattern: the app is hosted by the provider, the AI compute runs on the customer's own stack, and a human approves before anything is sent. The skills write into the customer's SPICED records; the agents reason over them.
